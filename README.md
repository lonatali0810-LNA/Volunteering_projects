**Волонтерский проект по изучению мотивации донорства**

**Файл:** volunteering_2026_04.ipynb

**Путь:** Volunteering_projects/projects/volunteering_2026_04.ipynb

**Инструменты:** SQL

**Описание проекта:**
**Цель проекта:** мотивировать людей становиться донорами и делать регулярные донации

**Задачи проекта:** volunteering
- определить какие факторы влияют на активность доноров 
- определить какими могут быть стратегии для мотивации донорства.

**План проекта:** основное внимание было уделено региональным особенностям, демографии (пол, возраст), роли бонусной системы, эффективности каналов привлечения (соцсети) и сравнению запланированных донаций с реальными. 
По итогу были прописаны рекомендации для формирования стратегии
